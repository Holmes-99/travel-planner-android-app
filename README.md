# ✈️ Travel Planner App
### ENCS515 Android Course Project — Birzeit University

A full-stack Travel Planner Android application built with Java and Spring Boot.

---

## 📱 Features

- 🔐 Login & Registration with encrypted passwords
- 🌍 Browse trips from REST API with search & filter
- ❤️ Add trips to favorites
- 📋 Make and view reservations
- ⭐ Special Section — Top Rated Trips (rating ≥ 4.5)
- 👤 Profile management
- 🛡️ Admin panel with full CRUD operations
- 🔔 Navigation Drawer with 8 sections

---

## 🛠️ Tech Stack

**Android (Frontend)**
- Java
- SQLite (DataBaseHelper)
- SharedPreferences
- RecyclerView
- Fragments & Navigation Drawer
- AsyncTask + HttpURLConnection
- Tween Animations

**Spring Boot (Backend)**
- Spring Data JPA
- MySQL
- RESTful API
